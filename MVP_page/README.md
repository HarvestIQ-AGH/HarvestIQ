# HarvestIQ — MVP landing page

Marketing / product-preview site for **HarvestIQ**: farm dashboards, alerts, data input, and an interactive map—plus a waitlist CTA. Bilingual UI (EN/PL) and light/dark theme.

## Tech stack

- **Next.js** (App Router) · **React** · **TypeScript**
- **Tailwind CSS** · **Radix UI**-based components (shadcn-style)
- **Leaflet** / **react-leaflet** for the farm map
- **next-themes**, **Vercel Analytics**, **react-hook-form** + **Zod**

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Production build: `npm run build` then `npm start`.
